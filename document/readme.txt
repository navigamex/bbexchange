1.删除不要的服务
2.创建一个仓库,初始化git
    git init  . 初始化这个项目, 把当前目录创建为一个git项目
    git add . ; 把当前目录下,除忽略文件全部添加
    git commit -am 这是备注,进行一次提交
    git remote add origin xxxx 添加远程仓库地址是xxx 上游是origin
    git branch 查看本地有哪些分支
    git branch master  基于当前分支,创建一个分支 叫master
    git checkout master  切换到master分支工作
    git push origin master -u 推送上去
